# 消防安全管理系统需求说明书

## 1. 项目概述

### 1.1 项目背景
随着安全生产要求的提高，企业需要一套完善的消防安全管理系统，以确保所有消防设备处于良好状态，及时进行维护和更换。由于消防设备种类多、分布广、有效期不同，手工管理难以保证及时性和准确性，因此需要开发一套专门的消防安全管理系统。

### 1.2 项目目标
开发一套集成化的消防安全管理系统，实现对微型消防站物资、灭火器和呼吸器等消防器材的全生命周期管理，特别是对即将到期或已过期物资的自动预警和通知功能，提高消防安全管理的效率和准确性。

## 2. 系统功能需求

### 2.1 用户管理模块
- **用户登录与认证**：支持用户名和密码登录，保护系统安全
- **用户角色管理**：支持管理员和普通用户两种角色
- **权限管理**：基于区域和操作类型的细粒度权限控制，包括查看、添加、编辑、删除权限
- **用户信息管理**：支持管理员添加、编辑、删除用户信息，以及重置密码

### 2.2 微型消防站物资管理
- **物资信息录入**：支持添加微型消防站物资信息，包括名称、型号、生产日期、安装位置等
- **物资信息查询**：支持按区域、物资名称等条件筛选查询
- **物资信息编辑**：支持对物资信息进行更新和维护
- **物资信息导出**：支持将物资信息导出为Excel格式

### 2.3 消防器材管理
- **器材信息录入**：支持添加灭火器、呼吸器等消防器材信息
- **器材信息查询**：支持多条件组合查询
- **器材信息编辑**：支持更新和维护器材信息
- **器材信息导出**：支持导出器材信息

### 2.4 有效期管理与预警
- **有效期规则设置**：支持设置不同类型物资的有效期规则
- **有效期计算**：基于生产日期和有效期规则自动计算到期日期
- **预警级别设置**：支持设置不同预警级别（已过期、30天内到期、60天内到期、90天内到期）
- **预警查询与显示**：直观显示所有预警物资，支持按预警级别、区域、负责人等筛选

### 2.5 邮件通知功能
- **邮件配置管理**：支持设置SMTP服务器、发件人信息等邮件发送配置
- **定时发送预警邮件**：支持设置定时任务，定期自动发送预警邮件
- **手动发送预警邮件**：支持手动触发发送预警邮件
- **负责人筛选**：支持按负责人筛选发送邮件
- **邮件日志记录**：记录所有邮件发送情况，包括成功和失败记录

### 2.6 区域与负责人管理
- **区域信息管理**：支持添加、编辑、删除区域信息
- **负责人信息管理**：支持设置各区域负责人信息，包括姓名、联系方式、邮箱等
- **区域权限关联**：将区域与用户权限关联，实现区域级的权限控制

### 2.7 统计分析功能
- **物资分布统计**：按区域统计物资分布情况
- **有效期预警统计**：统计不同预警级别的物资数量
- **邮件发送统计**：统计邮件发送成功率和数量

### 2.8 定时任务调度
- **任务配置管理**：支持配置定时任务的执行频率、时间等参数
- **任务执行记录**：记录定时任务的执行情况和结果

## 3. 非功能需求

### 3.1 性能需求
- **响应时间**：页面加载时间不超过3秒
- **并发处理**：支持多用户同时操作
- **数据处理能力**：支持处理至少10000条设备记录

### 3.2 安全需求
- **用户认证**：必须登录才能访问系统
- **权限控制**：基于角色和区域的访问控制
- **CSRF保护**：防止跨站请求伪造攻击
- **数据备份**：定期自动备份数据库

### 3.3 可用性需求
- **用户界面**：直观、易用的Web界面
- **操作指引**：提供必要的操作提示和帮助信息
- **错误处理**：友好的错误提示，不暴露系统细节
- **浏览器兼容性**：支持主流浏览器（Chrome, Firefox, Edge等）

### 3.4 可维护性需求
- **代码结构**：清晰的模块化设计
- **日志记录**：详细记录系统运行日志，便于排查问题
- **配置管理**：支持通过配置文件修改系统参数

## 5. 数据模型

### 5.1 主要数据实体
- **用户(User)**：用户账号信息
- **权限(Permission)**：用户操作权限
- **微型消防站物资(FireStation)**：微型消防站物资信息
- **消防器材(FireEquipment)**：灭火器、呼吸器等器材信息
- **有效期规则(EquipmentExpiry)**：不同物资的有效期规则
- **负责人(ResponsiblePerson)**：区域负责人信息
- **邮件日志(MailLog)**：邮件发送记录
- **调度配置(SchedulerConfig)**：定时任务配置
